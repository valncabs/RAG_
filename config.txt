Paso 1. Instalar virtualenv

En tu PowerShell ejecuta:

Comando

pip install virtualenv

Cuando termine debería decir algo como Successfully installed virtualenv.

Paso 2. Crear el entorno virtual

Aún dentro de C:\Users\velentina\Desktop\RAG, ejecuta:

Comando

python -m venv myvenv

Esto creará la carpeta myvenv.

Paso 3. Activar el entorno virtual

Primero ejecuta este comando (solo una vez):

Comando

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process

Si pregunta algo, escribe S y Enter.

Luego activa el entorno:

Comando

.\myvenv\Scripts\Activate

Con el entorno (myvenv) activado, ejecuta uno por uno estos comandos.
